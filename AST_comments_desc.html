VICIDIAL ADMIN: Scoring System Description

Point System (PS)

Description: This is the default soring when a campaign is created. Each question is scored PASS or FAILED. A pass is equal to a value when multiplied by the total number of questions would equal 100. A fail is equal to zero. Each equivalent score placed is then totaled. Total score is rated as follows:
  • 0 to 50 Points - FAIL
  • 51 to 100 Points - PASS
Note: A question marked Autofail fails the whole scoresheet regardless of any other score entered if the score given is a failing score. There can only be one (1) scoring system per campaign. Scoring system is editable during question editing.

Other Available Scoring Methods:

  • Averaging System(AV)
  • Percentile Sytem(PC)
